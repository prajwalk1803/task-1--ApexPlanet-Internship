# COMMITS_GUIDE.md

## Make 10 Git commits easily

Run these commands:

git init
git add .
git commit -m "Initial project setup"

# Commit 2
# Edit README.md slightly then:
git add README.md
git commit -m "Added documentation"

# Commit 3
# Update CSS colors or animations:
git add assets/css/style.css
git commit -m "Improved UI styling"

# Commit 4
# Update JS analytics:
git add assets/js/app.js
git commit -m "Added analytics dashboard"

# Commit 5
git add api/save_message.php
git commit -m "Added PHP save message API"

# Commit 6
git add api/get_messages.php
git commit -m "Added message fetching API"

# Commit 7
git add db/database.sql
git commit -m "Added MySQL schema"

# Commit 8
git add index.html
git commit -m "Improved portfolio sections"

# Commit 9
git add .github/workflows/pages.yml
git commit -m "Added GitHub Pages deployment workflow"

# Commit 10
git add .
git commit -m "Final task-1 submission update"
